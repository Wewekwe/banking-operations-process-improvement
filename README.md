## Project Overview
This project simulates a real-world banking KYC onboarding process to identify operational inefficiencies, SLA breaches, and compliance risks.

## Objective
To analyse onboarding data and propose process improvements that reduce delays and improve operational efficiency.

## Tools Used
- Microsoft Excel (Pivot Tables, Analysis)
- Business Analysis techniques
- Process improvement methodology

## Key Insights
- Missing and incorrect documents were the primary cause of SLA breaches
- Manual verification increased turnaround times
- Business customers required longer processing times

## Recommendations
- Digital document validation
- Risk-based KYC processing
- Improved customer communication

## Business Impact
- Estimated 25% reduction in processing time
- Improved SLA compliance
- Reduced rework and operational risk
